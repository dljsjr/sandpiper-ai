# Add storage status command showing current config and workspace health

sandpiper-tasks --dir . storage status would show:
- Current config source (.sandpiper-tasks.json / .sandpiper/settings.json / defaults)
- Effective mode (inline / separate-branch / external-repo)
- Workspace path and whether it exists
- VCS backend detected
- Whether .gitignore entry is present
- auto_commit / auto_push settings

Useful for diagnosing misconfiguration or missing workspace without having to read the config files manually.
